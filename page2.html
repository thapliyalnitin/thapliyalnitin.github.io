Hadoop Core Components & HDFS

Core Components

Hadoop has two main core components: HDFS (Hadoop Distributed File System) and MapReduce. HDFS is responsible for storing data, and MapReduce for processing it.

HDFS Architecture

HDFS uses a master-slave architecture. The NameNode is the master and stores metadata, while DataNodes handle actual data storage. HDFS is designed to store large files across multiple machines with redundancy and fault tolerance.
